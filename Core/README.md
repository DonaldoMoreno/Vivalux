# Core Module
Componentes principales del motor de Vivalux.

## Submódulos

### Render Engine
Motor de renderizado responsable de la visualización de contenido en superficies.

### Mapping Engine
Motor de mapeo para proyectar contenido en geometrías complejas.

### Media Manager
Gestor centralizado de archivos de media (imágenes, videos, assets).

### Scene System
Sistema de escenas para organizar y controlar elementos visuales.
